Overview
This project implements a lightweight DataFrame system in Java capable of:

Creating and storing tabular data, 
Filtering rows based on conditions, 
Grouping and aggregating data, 
Selecting and sorting columns, 
Joining two DataFrames, 
and Performing summary statistics

It is designed to mimic essential operations of analytics libraries such as pandas and R data.frames, but built entirely from scratch using Java OOP principles.

Features:
DataFrame Creation, 
Filtering, 
GroupBy + Aggregate, 
Sorting, 
Joining, 
and Column Selection
